## 二、排雷（负面清单，影子观察 P7）

七条红旗均为**"触发深挖"信号，非"确认造假"结论**；阈值按行业校准（毛利率行业内
比较）。出处：财报#3/#4/#7；〔验证〕`profit-vs-cash-and-fraud-screen`（干净公司
0 红旗 / 问题公司 8 红旗清晰分开）、`minority-interest-earnings-laundering`（乐视
归母 7.66 亿 vs 利润总额 0.4 亿 ≈19 倍红旗）、`goodwill-strip-revaluation`。

| 红旗 | 阈值 | 一句话依据 |
|---|---|---|
| R1 应收/营收 | >50% | 收入可能靠赊销堆出来（权责发生制：利润≠现金） |
| R2 商誉/归母净资产 | >30% | 虚资产 + 隐形杠杆（减值只减资产端，负债率必升） |
| R3 毛利率超行业中位 | >15pp 且行业样本≥5 | 好得不像真的（行业内校准——验证报告明确告诫） |
| R4 三费占比同比骤降 | >5pp | 费用腾挪粉饰净利润 |
| R5 归母 ≫/异号 净利润 | 比值>1.5 或异号 | 亏少数股东肥上市公司（乐视探针；快报是最后逃生窗口） |
| R6 经营现金流/净利润 | <30%（净利>0 时） | 利润+1.9 亿 vs 现金−4 亿的背离可精确勾稽 |
| R7 免税行业放大器 | 农林牧渔/软件/半导体等且已有红旗 | 造假税务成本低（獐子岛"扇贝游走"），只放大不单独定罪 |

行为：每调仓日全 universe 打分落 `quality_flag`；持仓/目标命中 ≥2 旗 → 计划
risk_hint 提示深挖；**不自动动仓、不剔除**。晋升为硬过滤的门槛见提案 P7。

## 六、运行模式（信号→计划→执行→复盘→迭代）

- **月频调仓 + 每日计划 + 盘中盯盘**：与"每期财报后重新决策"一致（长期持有是结果不是过程）。
- **财报季事件驱动重检**：fina_express（快报/预告）按公告日增量入库，操作计划每日
  扫描近 7 日新披露 → 戴维斯双杀预警（时效层内化）。
- **预测记分卡**（signal_scorecard / arb_scorecard）："不为结果负责的预测毫无价值"
  ——每个信号源按真实前瞻收益记账（投资篇#4 认知论内化）。
- **模型变更三关**：历史回测 → 影子并行 → 一键回退（evolution_roadmap 原则）；
  对应博主"数据为王、冷眼看数据"——规则以数据裁决，不以叙事裁决。
